Everything You Need to Know About WETRACK

Find answers to common questions about WETRACK's functionality, security, and usage for authorized users and monitored individuals.
What is WETRACK, and who is it for?

WETRACK is a web-based monitoring system designed for authorized agencies to track and manage individuals on probation or parole.

How does geo-fencing work?

Geo-fencing allows users to define virtual boundaries, and WETRACK sends alerts if these boundaries are breached.

Is the data stored in WETRACK secure?

Yes, WETRACK uses advanced encryption to safeguard sensitive data and comply with privacy regulations.

Can individuals being monitored access their data?

Yes, monitored individuals can view their compliance status, report issues, and access support resources.

What devices support WETRACK?

WETRACK is compatible with desktops, tablets, and mobile devices via modern web browsers.

Does WETRACK support real-time tracking?

Yes, WETRACK provides live GPS tracking and records historical data to ensure accurate compliance monitoring.
